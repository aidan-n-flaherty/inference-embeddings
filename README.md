### Encoding Logical Inference in Vector Embeddings

Create a python environment:
```
python -m venv .venv
source .venv/bin/activate
```

Install requirements:
```
pip install -r requirements.txt
```

Download the FactKG dataset here: [FactKG](https://drive.google.com/drive/folders/1q0_MqBeGAp5_cBJCBf_1alYaYm14OeTk)

Alternatively:
```
pip install gdown
gdown 1q0_MqBeGAp5_cBJCBf_1alYaYm14OeTk --folder
unzip factkg/factkg.zip
```
